# FinanceApp

Современное приложение для учёта личных финансов.

![Dashboard](screenshots/dashboard.png)

## ✨ Возможности

- ✅ Авторизация пользователей (логин/регистрация)
- ✅ Добавление доходов и расходов
- ✅ Категории транзакций
- ✅ Дашборд с общей статистикой
- ✅ Отчёты по периодам
- ✅ SQLite база данных (локальная)
- ✅ Красивый и удобный интерфейс WPF

## 🛠 Технологии

- **.NET 8** (или .NET 6/7 — уточни свою версию)
- **WPF** (Windows Presentation Foundation)
- **Entity Framework Core** + SQLite
- **MVVM** (в процессе перехода)
- **C#**
